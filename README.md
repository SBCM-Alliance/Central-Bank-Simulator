## 📖 概要

このシミュレーターは、中央銀行（日本銀行）総裁の視点から、金利操作が日本経済の「市場（数字）」と「実体（生活）」にどのような物理的衝撃を与えるかを可視化したツールです。

既存の経済ニュースが語る「GDP」や「日経平均」といった空中戦ではなく、住宅ローン、物価、実質賃金という「接地された痛み」を、動的なシミュレーションを通じて体験できます。

## 🚀 特徴

- **市場（期待）vs 実体（物理）の分離：** 反応の速い「株価・為替」と、遅れて致命的に効いてくる「物価・ローン」のギャップを図解。
- **ライブニュース同期（RSS2JSON）：** ロイター等の最新経済ニュースから、現在の日米金利差や世界情勢をリアルタイムに反映。
- **4つの市井のボイスボタン：** 「物価を下げろ」「ローンを守れ」といった庶民の切実な（しかし物理的に矛盾した）要求をボタン化。
- **マルチステークホルダー評価：** 政府（支持率）、経済界（投資）、家計（生存）の3つの視点からリアルタイムの悲鳴を表示。
- **「手取りマイナス」ロジック：** 利上げによる住宅ローン増が可処分所得を食いつぶし、家計が赤字転落する様子をマイナス棒グラフで描写。

## 🧪 物理的ロジック（SBCM的診断）

本シミュレーターは、以下の「統治工学」的バグを証明するために設計されています。

1.  **単一レバーの限界：** 「金利」という1つの変数で、数千の異なる利害を制御することの物理的無理性。
2.  **責任の分散：** プレイヤー（日銀）がブレーキをかけても、政府がバラマキ（加速）を行い、市場（投機）がノイズを撒き散らす「統治不能」の状態。
3.  **アース（接地）の欠如：** 通貨が実物経済から切り離され、単なる「期待」で空回りすることで発生する投機。

## 🛠️ 使い方

1.  `index.html` をブラウザで開きます。
2.  **「最新の市場状況に同期する」**ボタンを押し、現在の世界情勢を読み込みます。
3.  スライダーを動かして金利を調整するか、下部の要望ボタンを押して「特定の誰か」の要望に応えてみてください。
4.  グラフと「社会の悲鳴」欄を確認し、日本経済という巨船がどこへ向かっているかを観察してください。
